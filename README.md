# OpenClaw 配置指南

本仓库提供 OpenClaw 在 Windows 和 Debian 13 系统上的完整配置指南。

## 快速开始

查看完整指南: [OPENCLAW-config.md](./OPENCLAW-config.md)

## 文档目录

| 文档 | 说明 |
|------|------|
| [OPENCLAW-config.md](./OPENCLAW-config.md) | 完整配置指南 (推荐) |

## 主要内容

- Windows 10/11 安装
- Debian 13 安装
- NVIDIA API 配置 (免费额度)
- Ollama 本地模型集成
- 飞书聊天集成
- 局域网访问配置

## 版本要求

- OpenClaw: 2026.3.1+
- Node.js: 22.x+
