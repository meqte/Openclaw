# OpenClaw 部署与配置知识库

本仓库提供 OpenClaw 在 Windows 和 Debian 13 系统上的完整部署配置指南，以及其核心扩展技能 (Skills) 的管理手册。

## 快速开始

- 核心系统部署请查看: [OPENCLAW-config.md](./OPENCLAW-config.md)
- 智能体技能扩展请查看: [Skills.md](./Skills.md)

## 文档目录

| 文档 | 说明 |
|------|------|
| [OPENCLAW-config.md](./OPENCLAW-config.md) | 核心系统部署、模型接入 (NVIDIA/Ollama) 及飞书集成配置指南 |
| [Skills.md](./Skills.md) | 官方核心技能生态 (安全、搜索、工具、自动化) 的安装与 API 配置说明 |

## 主要内容

- Windows 10/11 / Debian 13 基础环境安装
- NVIDIA API / Ollama 本地模型集成
- 飞书聊天集成与局域网访问配置
- ClawHub 技能管理器使用指南
- 核心智能体技能 (如主动代理、自我改进、网页操作) 的批量部署

## 版本要求

- OpenClaw: 2026.3.1+
- Node.js: 22.x+
- ClawHub: 对应最新版
